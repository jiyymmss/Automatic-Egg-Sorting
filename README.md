# Automatic Egg Sorting & Pet Feeding

##  Overview
Automatically sorts eggs and send the record to Web Server

##  Components
- Arduino Uno
- PWM Motor Driver
- IR Sensors
- ESP8266
- SERVO MOTORS
- LOAD CELL
- HX711 AMPLIFIER
- 12V GEAR MOTOR
- SOLAR PANEL
- 12V RECHARGEABLE LEAD-ACID BATTERY

##  Features
- Automatically sorts egg
- Solar Powered Device
- Sends data to Web server for inventory
- Automatically Open the Pet Dispenser on the set schedule using Web Server

##  Challenges I Solved
- Building not strong enough so I seek help from experts to help me build the structure
- Sensor misreading → adjusted threshold values
